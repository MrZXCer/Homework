# Fibonacci

![](./screenshot.png)

## 环境

已测试环境：Ubuntu 16.04 x64 on Windows Subsystem for Linux

理论可行环境：*nix x64

## 说明

本代码支持**预先写定**的精度。可以通过修改1.asm的`PRECISION`宏来指定精度，理论上可以任意指定精度。

当精度为x时，支持十进制下最大10^(2x)的数。

默认情况下精度为40，支持F(x)<10^80

## 执行

`nasm -felf64 1.asm && ld 1.o && ./a.out`
